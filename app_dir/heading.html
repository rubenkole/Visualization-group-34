An interactive hospital visualisation

This dataset contains anonymised data from patients seen at the Hospital Israelita Albert Einstein, at Sao Paulo, Brazil, and who had samples collected to perform the SARS-CoV-2 RT-PCR and additional laboratory tests during a visit to the hospital.

You can use the buttons below to change the type of data you wish to view. The drop-down menus can be used to select the exact data you want to be presented in the graphs.
In case your colour vision is impaired, we have included the ability to select a different set of colours.
